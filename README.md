# ShaderLog

## Install

First install a console git client, [Node.js](https://nodejs.org/) and [tPrint](https://github.com/patriciogonzalezvivo/tprint) then do:

```bash
cd ~
git clone https://github.com/patriciogonzalezvivo/shaderLog.git
cd ~/shaderLog
npm install -l
```

## Run

```bash
cd ~/shaderLog
npm start
```
